GRUv2 - API FlightAware

Esta aplicação expõe uma API REST para consultar informações de voos, aeroportos e posições usando a FlightAware AeroAPI.

Endpoints disponíveis

  • GET /positions/:faFlightId — Consulta as posições de um voo pelo ID.
  • GET /flights/ — Busca um voo aleatório.
  • GET /flights/:faFlightId — Consulta detalhes de um voo pelo ID.
  • GET /airports/ — Lista os aeroportos mais movimentados.
  • GET /airports/:airport/arrivals — Lista chegadas em um aeroporto.
  • GET /airports/:airport/departures — Lista partidas em um aeroporto.
  • GET /airports/:airport/enroute — Lista voos em rota para um aeroporto.
  • GET /airports/:airport/scheduled — Lista voos agendados para um aeroporto.
  • GET /map/:faFlightId — Exibe o mapa de um voo.

Exemplo de uso

Para consultar posições de um voo, faça uma requisição GET para:

/positions/ID_DO_VOO

Configuração

  • PORT: Porta onde a aplicação será executada (padrão: 10000)
  • CACHE_TIME: Tempo de cache das consultas (padrão: 60s)
  • AEROAPI_KEY: Chave de API da FlightAware (deve ser definida nas variáveis de ambiente)
